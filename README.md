# 个人主页 🌟

一个生动活泼的卡通风格个人网站，用于展示个人技能、项目作品和博客笔记。

## 功能特点

- 🎨 **卡通风格设计** - 绿色 + 黄色配色，充满活力
- 📱 **响应式布局** - 完美适配手机、平板和桌面
- 📝 **博客系统** - 支持 Markdown 文章
- 🔍 **项目筛选** - 按类别筛选项目
- ✨ **交互动画** - 悬停效果、滚动动画

## 项目结构

```
portfolio/
├── index.html          # 主页（个人简介、技能、联系方式）
├── projects.html       # 项目作品展示页
├── blog.html           # 博客列表页
├── post.html           # 单篇文章模板
├── css/
│   └── style.css       # 主样式文件
├── js/
│   └── main.js         # 交互逻辑
├── blog/
│   └── learning-js.md  # 示例博客文章
└── README.md           # 项目说明
```

## 快速开始

1. 克隆项目
2. 用浏览器打开 `index.html`
3. 开始编辑内容

## 自定义

### 修改个人信息

编辑 `index.html` 中的内容：
- 姓名
- 个人简介
- 技能标签
- 联系方式

### 添加项目

在 `projects.html` 中添加新的项目卡片。

### 添加博客文章

1. 在 `blog/` 目录下创建 `.md` 文件
2. 在 `blog.html` 中添加文章卡片

## 部署

推荐使用 GitHub Pages 部署：

1. 创建 GitHub 仓库
2. 推送代码
3. 在 Settings > Pages 中启用 GitHub Pages
4. 选择 main 分支

## 技术栈

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+)
- Marked.js (Markdown 解析)
